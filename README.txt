
# EMS-Teknik – Time Registreringssystem

## Sådan bruger du det:

1. Upload indholdet af denne mappe til GitHub.
2. Aktiver GitHub Pages (Settings > Pages).
3. Opret en konto på https://supabase.com
4. Opret et projekt og kopiér din SUPABASE URL og public API key.
5. Erstat værdierne i `supabase.js` med dine egne.
6. Nu virker appen – og data bliver gemt online!

Brug evt. Netlify i stedet for GitHub hvis du ønsker nem adgang til PWA og forms.
